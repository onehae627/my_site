# my_site
 나의 포트폴리오 사이트


html,css,javascript를 사용해 구현했습니다.

favicon으로 웹브라우저 상단 탭 이미지를 삽입했습니다.

-자바스크립트를 사용해 스크롤을 내릴 때 홈 섹션의 요소들이 투명해지는 효과를 넣었습니다.

-반응형 웹 사이트로 구현했습니다.

-자바스크립트를 사용해 반응형 메뉴 아이콘을 눌렀을 때 메뉴 아이템이 나올 수 있게하고, 메뉴 아이템을 클릭했을 때 메뉴가 닫히도록 했습니다.

-자바스크립트를 사용해 arrow가 화면 높이에 따라 나타나고 사라지게 구현했습니다.

-css로 애니메이션 효과를 줘 눈을 재밌게 해봤습니다.

-메뉴를 누르면 해당 부분으로 이동하게 만들었습니다.

-arrow를 누르면 화면 상단으로 이동하게 했습니다.

-프로젝트 이미지를 누르면 해당 페이지로 이동하게끔 만들었습니다.

봐주셔서 감사합니다!!
